# T20-World-Cup-Cricket-Data-Analysis

This project analyzes T20 Cricket World Cup player data to identify the optimal playing XI using a full data pipeline —
from raw JSON web scraping to interactive Power BI dashboards, assisted by AI for KPI design and parameter scoping.

🔧 How I Built It:

📥 Data Collection — Scraped raw T20 World Cup data from the web, receiving it in JSON format and parsing it using Python.

🧹 Data Cleaning & Transformation — Cleaned, transformed, and structured raw JSON data into analytics-ready CSV files using Pandas.

📊 Data Visualization — Loaded CSVs into Power BI, created calculated columns and DAX measures to derive meaningful KPIs across 5 player categories.

🤖 AI-Assisted Design — Leveraged AI tools to define KPIs, parameter scoping, and dashboard logic for each player category — Proving that AI doesn't replace the analyst, it supercharges them.
